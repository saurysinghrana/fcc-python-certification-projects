# Scientific Computing with Python – Certification Projects

This repository contains the five certification projects completed as part of the **Scientific Computing with Python** certification offered by **freeCodeCamp**. The projects cover core Python concepts including string manipulation, time arithmetic, object-oriented programming, inheritance, and probability simulation.

---

## 📚 Projects

| Project                     | Description                                                                                                                       |
| --------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| **Arithmetic Formatter**    | Formats arithmetic expressions vertically while validating user input and producing neatly aligned output.                        |
| **Time Calculator**         | Calculates future dates and times by adding durations, supporting day transitions and optional weekdays.                          |
| **Budget App**              | Implements a budget management system using object-oriented programming, complete with transaction tracking and a spending chart. |
| **Polygon Area Calculator** | Models rectangles and squares using inheritance while calculating geometric properties and generating text-based representations. |
| **Probability Calculator**  | Estimates probabilities using Monte Carlo simulation by performing repeated random experiments on a virtual hat of colored balls. |
