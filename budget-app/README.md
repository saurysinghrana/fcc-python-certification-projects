# Budget App

A Python application that simulates a simple budgeting system using object-oriented programming. It allows users to manage multiple budget categories, record financial transactions, transfer funds, and visualize spending through a text-based bar chart. This project was completed as part of the **freeCodeCamp Scientific Computing with Python** certification.

---

## ✨ Features

* Create and manage multiple budget categories.
* Record deposits and withdrawals with optional descriptions.
* Transfer funds between budget categories.
* Automatically calculate category balances.
* Display neatly formatted transaction ledgers.
* Generate a percentage-based spending chart across multiple categories.

---

## 🎯 Project Objective

The objective of this project is to design and implement a budget management system using object-oriented programming principles. It demonstrates how classes, methods, and data structures can be used to model real-world financial operations while producing clear and formatted textual output.
