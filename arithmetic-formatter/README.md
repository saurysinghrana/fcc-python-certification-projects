# Arithmetic Formatter

A Python program that formats arithmetic problems vertically and displays them side-by-side according to standard arithmetic formatting rules. This project was completed as part of the **freeCodeCamp Scientific Computing with Python** certification.

---

## ✨ Features

* Arranges addition and subtraction problems vertically.
* Supports formatting up to five arithmetic problems simultaneously.
* Right-aligns operands for proper mathematical formatting.
* Optionally displays the calculated answers.
* Validates input and returns meaningful error messages for invalid expressions.

---

## 🎯 Project Objective

The objective of this project is to develop a function that formats arithmetic expressions into a vertically aligned layout while adhering to strict formatting and validation requirements. The project emphasizes clean string manipulation, conditional logic, and problem decomposition using Python.
