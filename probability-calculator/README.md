# Probability Calculator

A Python application that estimates probabilities using Monte Carlo simulation. The program performs repeated random experiments by drawing colored balls from a virtual hat without replacement and approximates the likelihood of specific outcomes. This project was completed as part of the **freeCodeCamp Scientific Computing with Python** certification.

---

## ✨ Features

* Create customizable hats containing colored balls.
* Draw random balls without replacement.
* Simulate thousands of probability experiments.
* Estimate probabilities using Monte Carlo simulation.
* Preserve the original hat during experiments through deep copying.
* Support custom expected outcomes and experiment parameters.

---

## 🎯 Project Objective

The objective of this project is to estimate probabilities through repeated random experiments instead of analytical calculations. It demonstrates simulation techniques, random sampling, object-oriented programming, and data processing using core Python.
