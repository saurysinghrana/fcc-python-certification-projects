# Time Calculator

A Python program that adds a specified duration to a given start time while accurately handling AM/PM conversion, day transitions, and optional weekdays. This project was completed as part of the **freeCodeCamp Scientific Computing with Python** certification.

---

## ✨ Features

* Adds a duration to a given start time.
* Correctly handles AM/PM conversion using a 12-hour clock.
* Calculates day transitions for durations spanning multiple days.
* Supports an optional starting day of the week.
* Displays `(next day)` or `(<n> days later)` when applicable.
* Produces correctly formatted, easy-to-read output.

---

## 🎯 Project Objective

The objective of this project is to build a time calculation function capable of accurately adding durations while accounting for day rollovers and optional weekday calculations. The project demonstrates logical problem-solving, time conversion techniques, and clean implementation using core Python.
