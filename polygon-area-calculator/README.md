# Polygon Area Calculator

A Python application that models rectangles and squares using object-oriented programming. It provides methods to calculate geometric properties, generate text-based shape representations, and determine how many shapes can fit inside another. This project was completed as part of the **freeCodeCamp Scientific Computing with Python** certification.

---

## ✨ Features

* Create and manage rectangle and square objects.
* Calculate the area, perimeter, and diagonal length of shapes.
* Generate text-based visual representations using `*` characters.
* Resize rectangles and squares through dedicated setter methods.
* Determine how many times one shape can fit inside another without rotation.
* Demonstrate inheritance by extending the `Rectangle` class with the `Square` class.

---

## 🎯 Project Objective

The objective of this project is to apply object-oriented programming concepts by designing reusable geometric classes. It demonstrates inheritance, method overriding, encapsulation, and mathematical computations while modeling real-world shapes in Python.
